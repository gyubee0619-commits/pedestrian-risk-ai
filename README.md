# Pedestrian Risk AI

## 프로젝트 소개

pedestrian Risk AI는 이미지에서 자동차, 오토바이, 자전거 등의 위험 객체를 인식하고, 이를 기반으로 보행자 관점의 위험 상황을 분석하기 위한 프로젝트입니다.
본 프로젝트는 YOLO 기반 객체 검출 기술을 활용하여 객체 인식 결과를 바탕으로 이후 위험도 판단 로직 및 화면 시각화 기능을 연결 하는 것을 목표로 합니다. 

## 프로젝트 목표
- 보행자 위험 상황과 관련된 객체를 탐지한다.
- 탐지된 객체의 종류, 위치, 크기 정보를 추출한다.
- 추출된 정보를 기반으로 위험도 판단 로직과 UI 화면 구성에 활용한다.

##팀 역할 분담
- 객체 인식 파트: 이미지에서 객체 탐지, 객체 종류/위치/크기 추출
- 위험도 로직 파트: 객체 인식 결과를 바탕으로 위험도 판단 규칙 구현
- UI/화면 파트: 객체 인식 결과와 위험도 결과를 시각화

##현재 구현 범위
- YOLO 기반 객체 인식
- 입력 이미지에서 객체 탐지
- 탐지된 객체의 종류(class), 위치(bounding box), 크기(size) 확인 가능
- 객체의 종류/위치 관계를 기반으로 한 위험도 계산
- 위험 상황 경고 로직
- 결과 시각화 UI

##데이터 구조
- 학습데이터는 dataset/폴더에 저장되어 있습니다.
- 데이터셋은 train, valid 폴더로 구성 되어 있으며, 각 폴더는 images,label를 포함합니다.
- 클래스 정보 및 데이터 경로는 data.yaml에 정의되어 있습니다.

## 입력 및 출력
1. 입력
- 테스트 이미지
- 학습된 모델 파일(best.pt)
2. 출력
- 탐지된 객체의 클래스
- 바운딩 박스 좌표
- 객체 크기 정보
- 위험도 표시
3. 예시
[ {"class_name": "pedestrian", "x1": 120, "y1": 80, "x2": 200, "y2": 300}, {"class_name": "car", "x1": 300, "y1": 150, "x2": 500, "y2": 350} ]
